Problemas críticos actuales
CRÍTICO Bug
PMIDs inventados en el reporte

El reporte PDF genera PMIDs aleatorios con Math.random() — son citas falsas. En una tesis esto es gravísimo. Las citas deben venir siempre de la API de PubMed.

CRÍTICO Lógica
Mapeo de enfermedades hardcodeado

La tabla de enfermedades (ABCA1→Tangier, SCN1A→Dravet) está escrita fija en el JS. Si el análisis devuelve genes nuevos, aparecen como "Requiere correlación" sin información real.

ALTO Datos
Solo 4 resultados de enriquecimiento

El CSV muestra 4 términos únicamente porque el análisis usa p-adj < 0.05 que con solo 5 genes raramente se alcanza. Falta incluir p-valor nominal y más bases de datos.

ALTO Fuentes
Búsqueda de genes incompleta

Solo combina TargetScan + miRTarBase. Falta Miranda, PicTar, DIANA-microT, TarBase — la intersección de múltiples predictores da mayor confianza estadística.

Mejoras de lo que ya existe
MEJORAR — Backend
Reemplazar PMIDs falsos por reales

El endpoint de PubMed ya existe en el código (get_pubmed_details) pero no siempre se usa en el reporte. Conectarlo correctamente a cada gen y pathway.

MEJORAR — Análisis
Relajar umbral de enriquecimiento

Con 5 genes, p-adj < 0.05 es demasiado estricto. Mostrar también p-valor nominal < 0.05 y ordenar por Combined Score. Incluir el campo "Overlap" en la tabla visible.

MEJORAR — UI
Dashboard de genes con más detalle

Los "gene pills" solo muestran el nombre. Agregar: al hacer clic en un gen, abrir un panel lateral con su función, expresión tisular, variantes clínicas y artículos.

MEJORAR — Tests
Ampliar la cobertura de tests

Solo hay 3 tests y uno de ellos (test_pipeline_request_validation) está vacío. Agregar tests de integración para cada endpoint y para el pipeline completo.

MEJORAR — Datos
Modo "umbral de consenso"

Actualmente la intersección es estricta (gen debe estar en TODOS los miRNAs). Agregar modo "≥N de M miRNAs" para detectar más genes con alta confianza pero no perfecta.

MEJORAR — Reporte
Reporte PDF con formato académico

El PDF actual es funcional pero no tiene formato APA/Vancouver para citas. Agregar sección de bibliografía real, resumen estructurado (Objetivo, Métodos, Resultados, Conclusión).
